# Navigator

**Benefits & Assistance Hub by HearthMind LLC**

Navigator helps users discover and track government benefits, veteran resources, and assistance programs. Designed with trauma-informed principles for neurodivergent users.

## Features

- **Benefits Guide** - Federal and state assistance programs with eligibility info
- **Resource Directory** - Veterans services, tech industry support, local resources
- **Progress Tracking** - Checklist system for application workflows
- **Timeline View** - Track deadlines and milestones

## Tech Stack

- Flask (Python)
- YAML-based configuration
- Vanilla HTML/CSS/JS frontend

## Running Locally

```bash
cd navigator
python -m venv venv
source venv/bin/activate  # or venv\Scripts\activate on Windows
pip install -r requirements.txt
python run.py
```

Then visit http://localhost:5000
